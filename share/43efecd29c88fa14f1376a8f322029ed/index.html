Big Data Analytics - Overview

Authors: Unknown

This document provides an overview of Big Data Analytics, focusing on the significant increase in data volume coupled with decreasing storage costs. It highlights the challenge of processing the vast amounts of data generated by private companies and research institutions, encompassing user interactions, business data, social media activity, and sensor data from devices like mobile phones and automobiles. Big Data Analytics is presented as the solution to this challenge, involving data collection from diverse sources, transformation of this raw data into consumable formats for analysts, and ultimately, the delivery of valuable data products to organizations. The core of the process lies in converting unstructured data from various sources into usable data products. Furthermore, the document introduces the concept of a data life cycle for Big Data Analytics, drawing parallels with the CRISP-DM methodology. This cycle, though not strictly linear, offers a framework for organizing the work required to derive insights from Big Data. The document explicitly references the Cross Industry Standard Process for Data Mining (CRISP-DM) as a foundational methodology for managing the complexities of Big Data Analytics. The CRISP-DM methodology, a well-established framework, is presented as a useful tool for structuring the entire process, emphasizing the interconnectedness of the various stages involved in extracting meaningful insights from large datasets. Key Concepts:

  • Big Data Analytics involves collecting, processing, and delivering data products for organizational use.
  • The Data Life Cycle provides a framework for organizing and delivering insights from Big Data.
  • CRISP-DM methodology is a cycle used to manage and analyze Big Data.
  • Exploratory Data Analysis (EDA) is crucial for understanding data.
  • Data preparation involves reshaping and preprocessing data for modeling.
  • Modeling focuses on trying different models to solve business problems.
  • Real-time data processing can avoid formal data storage.
  • Labeling examples by hand is crucial for supervised learning.
  • Big Data Analytics relies heavily on effective data collection.
  • Data collection is a key component of the Big Data cycle, leveraging sources like the Internet.
