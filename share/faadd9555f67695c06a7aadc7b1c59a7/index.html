Key Concepts:

  • Mean Squared Error (MSE) or L2 Loss: A loss function that calculates the average of the squared differences between predicted and actual values.
  • Mean Absolute Error (MAE) or L1 Loss: A loss function that calculates the average of the absolute differences between predicted and actual values.
  • Huber Loss or Smooth L1 Loss: A loss function that combines MSE and MAE, providing robustness to outliers.
