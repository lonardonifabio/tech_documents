VIP Cheatsheet: Supervised Learning

Authors: Afshine Amidi, Shervine Amidi

This document provides a concise cheatsheet introduction to supervised learning concepts, primarily focused on differentiating between discriminative and generative models. It outlines the core ideas behind building classifiers that predict outcomes (y) from input data (x). The key distinction lies in whether the model directly estimates P(y|x) (discriminative) or estimates P(x|y) to deduce P(y|x) (generative). The cheatsheet details common model types, including linear regression, logistic regression, Support Vector Machines (SVMs), Naive Bayes, and Neural Networks, categorized as either discriminative or generative. It introduces the concept of a hypothesis (hŒ∏) representing the chosen model, and explains how the model predicts output based on input data. Crucially, it defines a loss function (L(z,y)) as a measure of the difference between the predicted value (z) and the actual data value (y), listing common loss functions like least squared error, logistic loss, hinge loss, and cross-entropy loss. The document also touches upon cost functions and their role in optimization. The cheatsheet serves as a quick reference for understanding fundamental supervised learning principles and model types.
