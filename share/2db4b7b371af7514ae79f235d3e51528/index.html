FMAData Wrangling with pandas Cheat Sheet

Authors: pandas

This document provides a concise cheat sheet for data wrangling using the pandas library in Python. It focuses on fundamental operations for manipulating dataframes, primarily centered around the concept of 'tidy data'. The core principles outlined are that a tidy dataset should have each variable represented as a column and each observation as a row. The cheat sheet details several key functions and methods within pandas, including `pd.melt` for reshaping data by gathering columns into rows or spreading rows into columns, `pd.concat` for appending rows or columns of DataFrames, `df.pivot` for reshaping data based on specified columns and values, `df.sort_values` for ordering rows based on column values (ascending or descending), `df.rename` for renaming columns, `df.sort_index` for sorting the DataFrame index, and `df.drop` for removing columns. Furthermore, it demonstrates how to create DataFrames from lists of lists or dictionaries, specifying values for each column and row. The document emphasizes the intuitive nature of pandas when working with tidy data, leveraging vectorized operations automatically. The examples provided illustrate the practical application of these techniques, offering a quick reference for data wrangling tasks. The document is designed as a helpful guide for users new to pandas or those seeking a refresher on essential data manipulation techniques. It's a valuable resource for transforming and preparing data for analysis within a pandas environment.

Key Concepts:

  • Creating DataFrames in pandas
  • Tidy Data principles (variables in columns, observations in rows)
  • Reshaping data using melt and pivot
  • Appending DataFrames using concat
  • Sorting DataFrames by values using sort_values
  • Renaming columns using rename
  • Sorting DataFrames by index using sort_index
  • Dropping columns from DataFrames
  • Subset Observations (Rows) and Variables (Columns)
