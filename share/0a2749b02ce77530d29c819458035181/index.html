Python for Data Science Cheat Sheet

Authors: Datacamp

This document provides a concise cheat sheet for Python programming, specifically geared towards data science applications. It covers fundamental Python basics, including variable assignment, data types (strings, integers, floats, booleans), and list manipulation. The document details how to perform basic arithmetic operations, including addition, subtraction, multiplication, exponentiation, and modulo. It also explains the use of built-in functions like `str()`, `int()`, `float()`, and `bool()`. A significant portion of the cheat sheet focuses on list operations, demonstrating how to access elements using indexing and slicing, create lists, and modify lists through appending, removing, inserting, sorting, and reversing. The document includes examples for accessing elements within nested lists and demonstrates how to count the occurrences of an item within a list. Furthermore, it illustrates how to use the `index()` and `count()` methods for lists. The cheat sheet emphasizes the importance of understanding list indexing, which starts at 0, and provides clear examples for accessing elements at specific indices. The document is designed as a quick reference guide for data scientists learning or working with Python for data analysis and manipulation. It's a valuable resource for quickly recalling common Python syntax and operations related to data structures and basic calculations.

Key Concepts:

  • Variable assignment and initialization
  • Data types in Python (str, int, float, bool)
  • List manipulation (indexing, slicing, subsetting, appending, removing, inserting, sorting, reversing, extending, popping)
  • List of lists operations
  • String manipulation
  • Array creation and manipulation in NumPy
  • Data loading from text files (loadtxt, genfromtxt)
  • Saving arrays to disk (savetxt, save, savez)
  • Array copying (copy, view)
  • Array sorting
