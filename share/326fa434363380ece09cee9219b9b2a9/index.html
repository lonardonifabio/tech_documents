AWS Services Overview

Authors: nire0510

This document provides a concise overview of key Amazon Web Services (AWS) offerings. It details several core services, including VPC (Amazon Virtual Private Cloud), which allows for the creation of isolated cloud environments for launching AWS resources. AWS Direct Connect is highlighted as a service for establishing dedicated network connections between an organization's network and AWS, facilitating secure and reliable data transfer. Route 53, Amazon's scalable DNS and Domain Name Registration service, is also described. The document then delves into compute services like EC2 (Amazon Elastic Compute Cloud), offering resizable cloud computing capacity, and Elastic Beanstalk, an application container for simplified deployment and management. Furthermore, it covers serverless computing with AWS Lambda, which automatically manages compute resources in response to events. The excerpt also includes information on Amazon ECS (Elastic Container Service), enabling the management of Docker containers across EC2 instances, and AWS Mobile Hub for mobile app development and monitoring. Finally, it touches upon Device Farm for app testing on real devices within the AWS Cloud, SNS (Simple Notification Service) for message publishing, and Cognito for user identity and data synchronization. The document serves as a foundational introduction to a wide range of AWS services, catering to diverse application development and infrastructure needs. Key Concepts:

  • Amazon Virtual Private Cloud (VPC) - private, isolated cloud environment.
  • AWS Direct Connect - dedicated network connection to AWS.
  • Amazon Route 53 - scalable DNS and Domain Name Registration service.
  • Amazon Elastic Compute Cloud (EC2) - resizable compute capacity in the cloud.
  • AWS Elastic Beanstalk - application container for deployment and management.
  • AWS Lambda - compute service triggered by events, automatically managing resources.
  • Amazon ECS - running and managing Docker containers across EC2 instances.
  • AWS Mobile Hub - building, testing, and monitoring mobile apps.
  • AWS Device Farm - testing apps on real devices in the AWS Cloud.
  • Amazon Simple Notification Service (SNS) - publishing messages to subscribers.
