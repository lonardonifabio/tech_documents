Programming Computer Vision with Python

Authors: Jan Erik Solem

This document, "Programming Computer Vision with Python," is a pre-production draft focusing on practical implementation using Python. It provides a foundational guide for learning computer vision techniques. The book begins with an introduction to the core concepts and tools. Specifically, it covers the use of the Python Imaging Library (PIL), Matplotlib, and NumPy for image handling and processing. A significant portion is dedicated to exploring various image descriptors, including the Harris corner detector and the Scale-Invariant Feature Transform (SIFT). The book delves into matching geotagged images, and introduces image to image mappings using homographies. The content is structured around practical examples and techniques, aiming to equip readers with the skills to develop computer vision applications. The draft is released under the Creative Commons Attribution-Noncommercial-No Derivative Works 3.0 United States License, emphasizing non-commercial use and prohibiting derivative works. The book's structure includes a preface, prerequisites, an introduction to computer vision, and detailed sections on image handling, descriptors, and mappings. It's a resource for those seeking a hands-on approach to computer vision using Python, with a strong emphasis on practical implementation and understanding of key algorithms.
