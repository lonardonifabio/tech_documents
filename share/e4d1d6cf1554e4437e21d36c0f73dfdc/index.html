Gemini API: Quickstart with Python

Authors: Google LLC

This document provides a quickstart tutorial for utilizing the Gemini API using Python. It guides users through the initial steps of accessing and experimenting with Google's Gemini large language models. The tutorial focuses on practical application, enabling users to generate text responses from both text and multimodal inputs (including images). A key component of the quickstart is the ability to engage in multi-turn conversations, leveraging the chat functionality of the Gemini API. Furthermore, the document introduces the concept of embeddings and their application with large language models. The tutorial is designed to be easily accessible, offering two execution options: running directly within Google Colab, eliminating the need for local environment setup, or executing the notebook locally, provided the user has Python 3.9 or higher and a Jupyter installation. The quickstart emphasizes a hands-on approach, allowing users to quickly understand and implement the Gemini API's capabilities. The provided links facilitate direct access to the Colab notebook and the underlying documentation, ensuring a seamless learning experience.
