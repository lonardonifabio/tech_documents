Key Concepts:

  • The linear regression model is a statistical method used to model the relationship between a dependent variable and one or more independent variables.
  • Correlation and regression are distinct statistical concepts that are often confused.
  • The geometrical representation of the Linear Regression Model provides a visual understanding of the model's relationship.
  • Linear relationship between two variables is suitable for linear regression models.
  • Endogeneity refers to a link between independent variables and errors, leading to omitted variable bias.
  • Omitted variable bias occurs when a relevant variable is forgotten, causing correlation between independent and dependent variables.
