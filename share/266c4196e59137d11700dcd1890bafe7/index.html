Key Concepts:

  • Convolutional Neural Networks utilize filters and layers.
  • Recurrent Neural Networks are designed to handle long-term dependencies.
  • Word embeddings represent words in a vector space.
  • A language model predicts the next word in a sequence.
  • IoU is a function quantifying the overlap between predicted and actual bounding boxes.
  • YOLO is an object detection algorithm that divides images into a grid and predicts bounding boxes for each cell.
  • Non-max suppression removes duplicate overlapping bounding boxes based on prediction probability and IoU.
  • Perplexity is the inverse probability of the dataset normalized by the number of words.
  • Beam search is a heuristic search algorithm for finding the likeliest sentence given an input.
  • Beam width is a parameter in beam search, affecting performance and computational cost.
