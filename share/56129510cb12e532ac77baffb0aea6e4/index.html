Building Blocks of Machine Learning Methods

Authors: Unknown

This book provides a foundational understanding of common machine learning methods, presented as a practical 'toolbox' for machine learning engineers. It focuses on delivering the core concepts and derivations of these methods, aiming to equip readers with the ability to understand and implement them from first principles. The book is structured around individual machine learning methods, with each chapter dedicated to a single method or a group of related methods. A key element of the book's approach is to break down each chapter into three sections: a 'concept' section that introduces the methods conceptually and derives their results mathematically; a 'construction' section that demonstrates how to build the methods; and a practical approach to learning by deriving the methods from scratch, both theoretically and through code. The book intends to foster a deep understanding of the underlying principles, rather than simply presenting pre-built solutions. The structure is designed to allow readers to gain a comfortable familiarity with these methods, providing them with the necessary tools for a variety of tasks within the field of machine learning. The book's goal is to empower engineers to not just use existing methods, but to truly understand and manipulate them.

Key Concepts:

  • This book covers the most common methods in machine learning.
  • Each chapter corresponds to a single machine learning method or group of methods.
  • The purpose of this book is to provide those derivations.
  • Understanding how algorithms work intuitively.
  • Different algorithms create models.
  • The elements of various models.
  • Knowledge of calculus, probability, and linear algebra is required for understanding the book's concepts.
  • An observation is a collection of predictors and target variables.
  • Training data sets are used to build machine learning models.
  • Validation data sets are used to compare multiple models built on the same training data set.
