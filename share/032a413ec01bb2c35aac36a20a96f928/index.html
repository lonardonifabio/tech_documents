Power BI for Intermediates

Authors: Soheil Bakhshi, Bruce Anderson

This document, "Power BI for Intermediates," is a step-by-step training guide designed for users with some existing Power BI knowledge. It focuses on utilizing best practice methodologies within the Power BI environment. The guide is structured into two main parts, 'Part A' and 'Part B,' covering fundamental aspects of the tool. Part A begins with an introduction to Power BI Desktop and explores connecting to various data sources, including data sources and connection modes like Import, Direct Query, and Mixed Mode. It details the process of data preparation using Power Query, specifically focusing on splitting, renaming, adding, and removing columns, as well as changing data types. The document then delves into data modelling, emphasizing the importance of relationships between data tables and identifying key columns, particularly primary and foreign keys. The guide provides practical steps and explanations for each stage of the process, aiming to equip users with the skills to effectively build and manage Power BI reports and dashboards. The content is organized with detailed sections and sub-sections, offering a structured learning experience. The document is produced by Theta Systems Limited and includes contact information for inquiries. It is a comprehensive resource for intermediate Power BI users seeking to enhance their skills and understanding of best practices.

Key Concepts:

  • Step-by-step training guide for Power BI
  • Connecting to various data sources using Power BI
  • Data preparation techniques using Power Query (Splitting, Renaming, Adding, Removing, Changing Data Types)
  • Data modelling and relationships in Power BI
  • Identifying primary and foreign keys in Power Query
  • Data preparation involves profiling and transforming data.
  • Power Query interface allows for data transformations and creation of a data model.
  • The application ribbon contains options and settings for Power Query.
