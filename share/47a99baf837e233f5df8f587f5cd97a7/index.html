Efﬁ cient Learning Machines

Authors: Mariette Awad, Rahul Khanna

Efﬁ cient Learning Machines is a comprehensive exploration of machine learning techniques designed for intermediate to advanced users. The book delves into the core concepts of machine learning, covering key areas such as knowledge discovery, classification, genetic algorithms, neural networking, kernel methods, and biologically-inspired approaches. Awad and Khanna adopt a synthetic approach, meticulously blending theoretical explanations with practical design principles and real-world applications. A central focus is on equipping engineers, students, and system designers with the skills to develop and implement efficient machine learning systems. The book emphasizes a hands-on learning experience through detailed analysis of sample algorithms, allowing readers to understand how machine learning technology can be applied to solve specific problems. It guides readers through recognizing and analyzing relevant problems, implementing standard solutions to illustrative examples, and ultimately, designing novel systems and solutions. Crucially, the text acknowledges the significant influence of advancements in computing infrastructure – including improvements in performance, storage, memory, unstructured information retrieval, and cloud computing – alongside the rise of big data analytics and new machine learning paradigms. The authors position these developments within a conceptual framework, providing a holistic understanding of the field. The book's structure is geared towards practical application, fostering a deep understanding of the underlying principles and methodologies of efficient machine learning, making it a valuable resource for those seeking to contribute to the design and development of cutting-edge machine learning systems. It bridges the gap between theoretical knowledge and practical implementation, preparing readers to tackle complex challenges in the rapidly evolving landscape of data science and artificial intelligence.

Key Concepts:

  • Cost-effective alternatives to traditional methods for extracting relationships between information and data.
  • Predicting future events by processing existing information to train models.
  • Design and creation of new and more efﬁ cient machine learning systems.
  • Nonparametric models are characterized by an infinite set of parameters and lack finite parameter expressions.
  • OLAP is an approach for resolving multidimensional analytical queries using indexed data.
  • Supervised learning involves extracting associations between attributes and a designated dependent attribute using a training dataset.
  • Reinforcement learning seeks past actions it explored that resulted in rewards.
  • The policy is a key component of an RL-agent that maps the control-actions to the perceived state of the environment.
  • The critic represents an estimated value function that criticizes the actions that are made according to existing policy.
  • The reward function estimates the instantaneous desirability of the perceived state of the environment for an attempted control-action.
