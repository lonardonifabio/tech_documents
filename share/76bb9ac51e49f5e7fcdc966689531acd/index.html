This document introduces Earned Value Management (EVM), a project management technique designed to comprehensively monitor and report on project performance. EVM integrates scope, schedule, and cost into a single system, providing a robust framework for tracking progress and identifying potential issues. The core of EVM revolves around tracking actual performance against planned values, utilizing metrics like Earned Value (EV), Cost Variance (CV), and Schedule Variance (SV) to assess project health. Key Concepts:

  • Earned Value (EV): A measure of the value of work completed.
  • Planned Value (PV): The budgeted cost of work scheduled to be completed.
  • Actual Cost (AC): The actual cost incurred for work completed.
  • Schedule Variance (SV): The difference between the budgeted and actual schedule.
  • Cost Variance (CV): The difference between the budgeted and actual costs.
  • S-Curve Report: A graphical representation of project performance over time.
