R Programming Cheat Sheet: String Manipulation

Authors: arianne Colton, Sean Chen

This document provides a concise cheat sheet for basic string manipulation techniques within the R programming language, specifically targeting R version 3.0 and above. It covers fundamental operations including string concatenation using the `paste()` function, handling separators, splitting strings with `str_split()`, extracting substrings with `str_sub()`, and searching for patterns within strings using `str_detect()`. The guide also addresses essential R concepts such as data types (including 64-bit integers and case sensitivity), R indexing (starting from 1), and the importance of function naming to avoid conflicts when multiple packages utilize the same function name. Furthermore, it highlights the use of `help()` and `?` for accessing documentation, the `apropos()` function for searching by partial names, and the `example()` function for accessing illustrative examples. Crucially, the document emphasizes the existence of hidden objects (those starting with a period) that are not visible through `ls()` and provides guidance on memory management using `gc()` to release unused memory. The guide also stresses the importance of understanding the environment in which R operates, particularly regarding function naming and potential conflicts between packages. The document serves as a quick reference for R users seeking to efficiently work with strings and understand core R programming concepts. It is designed to be a practical tool for beginners and experienced R programmers alike.
