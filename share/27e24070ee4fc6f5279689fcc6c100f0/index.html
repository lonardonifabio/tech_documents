Key Concepts:

  • AWS stands for Amazon Web Services, a platform providing on-demand resources for hosting web services.
  • EC2 is Elastic Compute Cloud, a service for running virtual machines.
  • S3 is Simple Storage Service, a storage service for storing data.
  • Key-pairs are secure login information for instances using public and private keys.
  • Pricing models for EC2 instances include on-demand, reserved, spot, scheduled, and dedicated.
  • EBS volumes are persistent volumes attached to instances, preserving data even when instances are stopped.
  • Security groups act as firewalls controlling instance traffic.
  • Shared AMIs are AMIs created and made available for use by others.
  • Application Load Balancer uses dynamic port mapping and multiple listeners.
  • Classic Load Balancer uses one port one listener.
