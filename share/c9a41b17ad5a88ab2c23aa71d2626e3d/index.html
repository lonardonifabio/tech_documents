Data Prep with SQL - Quick Reference

This document provides a quick reference guide for data preparation tasks using SQL. It covers several key aspects of data profiling, validation, and cleaning. The document focuses on common SQL queries and techniques used to assess and transform datasets. Specifically, it details methods for dataset profiling, including volume and velocity analysis using `COUNT(*)` and `GROUP BY` clauses, respectively. It also includes sections on attribute selection, identifying incomplete records based on null values, and validating attribute ranges using `MIN()`, `MAX()`, and `AVG()`. The guide offers techniques for examining data types through `information_schema`, detecting outliers using standard deviation and confidence intervals, and analyzing data distributions with `WIDTH_BUCKET`. Furthermore, it provides instructions on handling missing values through `COALESCE()` and `REPLACE()` functions, along with methods for correcting incorrect values. The document also includes examples of standardization, outlier treatment (quantitatively), and data transformation techniques like casting, replacing, and scaling. The overall goal is to equip users with a concise set of SQL commands and strategies for preparing data for analysis or modeling. The document is geared towards individuals involved in data analysis, data science, or data engineering roles who require a readily accessible reference for common data preparation tasks.

Key Concepts:

  • Dataset Profiling: Techniques for examining dataset characteristics like volume, velocity, attributes, and data types.
  • Data Type Selection: Utilizing information_schema to retrieve table column definitions.
  • Missing Value Handling: Strategies for dealing with missing data, including COALESCE, AVG OVER(), and replacing with default values.
  • Outlier Detection: Identifying values significantly different from the norm using standard deviation.
  • Scaling Techniques: Normalizing data by dividing by the maximum value within a partition.
  • Data Transformation: Applying functions like REPLACE, CAST, UPPER, and AVG OVER to modify and derive data.
