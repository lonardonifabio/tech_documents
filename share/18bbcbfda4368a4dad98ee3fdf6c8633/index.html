Pandas Mastery Workshop - Data Reading with Pandas

Authors: Enthought, Inc.

This document excerpt promotes the 'Pandas Mastery Workshop' offered by Enthought, Inc. It focuses on utilizing Pandas for data reading and writing, specifically detailing the various `pd.read_*` methods available for parsing data from different file types (e.g., CSV, HTML). Key aspects covered include reading data from web tables using `read_html()`, reading data from text files with options like `sep`, `header`, `skiprows`, `skipfooter`, `index_col`, `parse_dates`, and `na_values`. The excerpt highlights the flexibility of Pandas in handling diverse data formats and provides examples of common usage patterns. It also explains how to parse dates within the DataFrame and demonstrates how to handle missing values using `na_values`. The document serves as a brief introduction to the workshop's content and the core functionalities of Pandas for data manipulation.

Key Concepts:

  • Data reading with Pandas using `pd.read_*` methods.
  • Reading data from CSV files using `read_table`.
  • Reading data from HTML using `read_html`.
  • Parsing data with `parse_dates` to handle dates.
  • Handling missing values with `na_values`.
  • String manipulation with .str methods in Pandas Series
  • Regular expression extraction and searching
  • Data type conversion to Datetime objects
  • Creating date ranges with pd.date_range and pd.period_range
  • Resampling time series data with resample and groupby
