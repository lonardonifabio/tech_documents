Kubernetes Cheat Sheet

Authors: Linuxacademy

This document is a Kubernetes cheat sheet, providing a concise overview of common commands and concepts for managing Kubernetes clusters. It focuses on practical commands used for inspecting and interacting with various Kubernetes resources, including Nodes, Pods, Deployments, Services, and DaemonSets. The cheat sheet demonstrates how to retrieve information about these resources using `kubectl`, such as getting node details, pod status, deployment configurations, service information, and DaemonSet details. It includes commands for displaying resource information in different formats (wide, YAML, JSON), filtering resources based on their status or labels, and retrieving specific details like node addresses, pod names, and deployment configurations. The document is intended as a quick reference guide for users who are new to Kubernetes or need a reminder of frequently used commands. It highlights the core functionalities of `kubectl` and provides a starting point for exploring the Kubernetes ecosystem. The document is sourced from Linuxacademy and serves as a foundational resource for understanding and managing Kubernetes deployments.

Key Concepts:

  • Kubernetes is a platform for managing containerized workloads.
  • Kubernetes orchestrates computing, networking, and storage to provide seamless portability across infrastructure providers.
  • kubectl is a command-line tool for interacting with Kubernetes objects (pods, nodes, deployments, services, namespaces, daemonsets).
