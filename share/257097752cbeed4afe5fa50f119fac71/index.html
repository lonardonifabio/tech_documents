How to Choose the Right Data Visualization

Authors: Mike Yi

This document provides a foundational guide on selecting appropriate data visualizations for effective data analysis. It emphasizes the crucial role of visualizations in summarizing large datasets and highlights the importance of considering the intended purpose and data type when choosing a chart. The core principle revolves around defining the 'role' a visualization should fulfill, with common roles including illustrating changes over time, representing part-to-whole relationships, depicting flows and processes, analyzing data distributions, comparing values across groups, examining relationships between variables, and visualizing geographical data. The document stresses that the selection process hinges on understanding whether the data is categorical, numeric, or a combination of both. It advocates for a flexible approach, acknowledging that deviating from standard visualization modes can unlock further insights. The book is structured around these visualization roles, offering multiple chart types to accommodate diverse data types and analytical subtasks. Ultimately, the guide serves as a general starting point, encouraging experimentation and creative thinking to maximize the value derived from data visualization. Key Concepts:

  • Data visualizations are a vital component of a data analysis, as they have the ability to efficiently summarize large amounts of data through a graphical format.
  • Choosing the right way to represent your data using one of these visualizations is a key part of the analysis process.
  • The type of chart you use will depend on if the data is categorical, numeric, or some combination of both.
  • Experimenting with visualizations can lead to additional insights.
  • Data visualization techniques for understanding data distribution.
  • Use of bar charts to represent categorical data and their associated counts.
  • Histograms for visualizing continuous numeric data through binning.
  • Density curves as an alternative to histograms for continuous numeric data.
  • Box plots for comparing distributions between groups and identifying data ranges.
