Data Analysis with PANDAS Cheat Sheet

Authors: arianne Colton, Sean Chen

This document provides a concise cheat sheet for using the Pandas library in Python for data analysis. It focuses on two fundamental data structures: Series and DataFrame. Series are described as one-dimensional, array-like objects that can hold data and associated labels (indexes). The creation of a Series is demonstrated using `pd.Series`, allowing for custom index assignment or the use of a default integer index. The document highlights methods for accessing Series values, such as `values` and indexing with square brackets (`series1['a']` or `series1[['b','a']]`). It also introduces the concept of Series as a fixed-length, ordered dictionary, emphasizing their compatibility with functions expecting dictionary inputs. The document then introduces the DataFrame, a two-dimensional data structure resembling a dictionary of Series, and demonstrates its creation from lists or NumPy arrays. DataFrame creation is shown with and without specifying column order, and with custom index labels. The document emphasizes that DataFrames can be created from nested dictionaries. The core concepts presented are designed to quickly familiarize users with the basic operations and structures available within Pandas, facilitating efficient data manipulation and analysis. The document provides a practical guide for quickly learning the fundamentals of Pandas data structures.

Key Concepts:

  • Series: A one-dimensional array-like object with associated data labels (index).
  • DataFrame: A two-dimensional tabular data structure, often considered a dictionary of Series.
  • Index: The label associated with each element in a Series or DataFrame.
  • Data Alignment: Series can be auto-aligned in arithmetic operations based on their indices.

  • Grouping by multiple keys

  • Converting Groups to Dict
  • Grouping with functions
  • Resampling: The process of aggregating time series data based on a fixed frequency.
