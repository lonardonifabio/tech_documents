Booke h Cheat Sheet: Python for Data Science

Authors: Bryan Van de Ven

This document provides a concise cheat sheet for using Booke h, a Python interactive visualization library, for data science. It focuses on creating visualizations with Booke h, particularly for large datasets rendered in modern web browsers. The core of Booke h's functionality revolves around two main components: data and glyphs. The document outlines the basic steps to create plots using the `bokeh.plotting` interface, which involves preparing data (using Python lists, NumPy arrays, Pandas DataFrames, or other sequences of values), creating a new plot, adding renderers for the data with visual customizations, specifying where to generate the output, and showing or saving the results. Under the hood, Booke h converts the data to Column Data Sources. The example code snippet demonstrates the basic usage of Booke h, including importing necessary modules (`figure` and `output_file`, `show`) and creating a simple line plot. The example code shows how to define the plot title and axis labels. The document serves as a quick reference for data scientists who want to leverage Booke h's capabilities for efficient and interactive data visualization. It's a foundational guide for getting started with Booke h and understanding its core principles.

Key Concepts:

  • Bokeh plotting interface for creating visualizations.
  • Data preparation (Python lists, NumPy arrays, Pandas DataFrames).
  • Creating plots with the Bokeh plotting interface (prepare data, create plot, add renderers, specify output, show/save results).
  • Column Data Sources (automatic conversion of data).
