Power BI for Intermediates

Authors: Soheil Bakhshi, Bruce Anderson

This document, "Power BI for Intermediates," is a step-by-step training guide designed for users with some existing Power BI knowledge. It focuses on utilizing best practice methodologies within the Power BI environment. The guide is structured into two main parts, 'Part A' and 'Part B,' covering fundamental aspects of the tool. Part A begins with an introduction to Power BI Desktop and explores connecting to various data sources, including data sources and connection modes like Import, Direct Query, and Mixed Mode. It details the process of data preparation using Power Query, specifically focusing on splitting, renaming, adding, and removing columns, as well as changing data types. The document provides a thorough explanation of transformation steps within Power Query. Part B delves into data modelling, specifically examining relationships within data modelling and identifying key columns, including primary and foreign keys. The guide emphasizes practical application and best practices. The document is authored by Soheil Bakhshi (Microsoft MVP) and Bruce Anderson. It's a comprehensive resource for intermediate Power BI users seeking to enhance their skills in data preparation and modelling. The content is organized into a logical progression, starting with the basics and building towards more advanced techniques. The inclusion of a Microsoft MVP author adds credibility to the training material. The document's focus on best practices ensures users learn efficient and effective methods for working with Power BI.
