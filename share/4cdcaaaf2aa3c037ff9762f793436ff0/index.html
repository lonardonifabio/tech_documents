VIP Cheatsheet: Machine Learning Tips

Authors: Afshine Amidi, Shervine Amidi

This document provides a concise cheatsheet focused on key metrics used in evaluating classification models within the context of machine learning. It specifically addresses the assessment of model performance, outlining essential concepts and formulas. The core of the document centers around the confusion matrix, a fundamental tool for understanding classification errors. It defines the confusion matrix and explains the terms True Positives (TP), False Negatives (FN), True Negatives (TN), and False Positives (FP). The document then details several primary metrics, including accuracy, precision, recall, sensitivity, and specificity, providing their respective formulas and interpretations. Accuracy is defined as the overall performance of the model, calculated as TP + TN / (TP + TN + FP + FN). Precision measures the accuracy of positive predictions (TP / (TP + FP)), while recall (also known as sensitivity) assesses the model's coverage of actual positive samples (TP / (TP + FN)). Specificity, conversely, measures the coverage of actual negative samples (TN / (TN + FP)). The F1-score is introduced as a hybrid metric particularly useful for handling unbalanced datasets, calculated as 2TP / (2TP + FP + FN). Furthermore, the document explains the Receiver Operating Characteristic (ROC) curve and its associated metrics: True Positive Rate (TPR), which is equivalent to Recall, and False Positive Rate (FPR), which is equivalent to 1 - Specificity. The document serves as a quick reference guide for machine learning practitioners seeking to understand and interpret these critical performance indicators. The focus is on providing a clear and accessible explanation of these metrics, facilitating effective model evaluation and selection.
