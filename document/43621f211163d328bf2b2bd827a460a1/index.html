Big Data Analytics - Overview

Authors: Unknown

This document provides an overview of Big Data Analytics, focusing on the significant increase in data volume coupled with decreasing storage costs. It highlights the challenge of processing the vast amounts of data generated by private companies and research institutions, encompassing user interactions, business data, social media activity, and sensor data from devices like mobile phones and automobiles. Big Data Analytics is presented as the solution to this challenge, involving data collection from diverse sources, transformation of this raw data into consumable formats for analysts, and ultimately, the delivery of valuable data products to support organizational decision-making. The document emphasizes the core process of converting unstructured data into usable data products. Furthermore, it introduces the concept of a data lifecycle for Big Data Analytics, drawing parallels with the traditional CRISP-DM methodology. This lifecycle is described as non-linear and iterative, reflecting the interconnected nature of the various stages involved in extracting meaningful insights from large datasets. The document suggests that the CRISP-DM methodology, a cross-industry standard process for data mining, can be used as a framework to organize and manage the work required to achieve these goals. The document implicitly advocates for a structured approach to Big Data Analytics, leveraging established methodologies like CRISP-DM to ensure effective data processing and insight generation. The focus is on transforming raw data into actionable intelligence, a key aspect of modern data-driven organizations.
