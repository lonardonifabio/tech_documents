Machine Learning Cheatsheet

Authors: Rémi Canard

This document presents a concise cheatsheet for machine learning algorithms, drawing inspiration from resources like MachineLearningMastery, ML Pros & Cons of HackingNote, and The Probability Cheatsheet of W. Chen. It provides a foundational overview of key concepts and algorithms, categorized primarily by their approach – parametric (linear) versus non-parametric (nonlinear). The core distinction lies in the assumptions made about the target function 'f' mapping input variables 'X' to output variable 'Y' with an associated error 'e'. Linear algorithms simplify this mapping to a known linear combination, focusing on learning coefficients, while non-parametric algorithms offer greater flexibility, allowing them to learn any functional form from the training data. The document highlights the importance of understanding the bias-variance trade-off, a fundamental concept in supervised learning. Bias represents the simplifying assumptions made to facilitate learning, while variance reflects the model's sensitivity to changes in the training data. The cheatsheet emphasizes that supervised learning methods predict 'Y' from 'X' using labeled data, whereas unsupervised learning aims to discover inherent structure within unlabeled data. The document serves as a quick reference guide for understanding the fundamental differences between these approaches and the core concepts driving model selection and performance evaluation. It's designed to provide a starting point for exploring the broader landscape of machine learning algorithms and their respective strengths and weaknesses.
