Data Science Cheatsheet

Authors: Maverick Lin

This document is a concise cheatsheet outlining the fundamental concepts of data science. It defines data science as a multi-disciplinary field integrating computer science, statistics/machine learning, and data analysis to derive insights from vast data sets. The document highlights two primary research paradigms: hypothesis-driven, where the research question dictates the data needed, and data-driven, where the data itself reveals potential problems. A core principle emphasized is continuous questioning and curiosity regarding the data and potential actions derived from it. It clearly distinguishes between structured and unstructured data, defining structured data as having predefined formats like tables and relational databases, contrasting it with unstructured data which lacks such structure and includes formats like text, images, and audio. The document further categorizes data into quantitative (numerical, such as height and weight) and categorical (labeled groups, like race and sex) types. The concept of 'Big Data' is introduced, characterized by massive volumes, variety, and velocity, exceeding the capacity of a single machine. Common data formats like CSV, XML, SQL, and JSON are listed, alongside data sources including companies, APIs, government data, academic research, and web scraping.
