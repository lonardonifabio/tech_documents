Deep Learning With Python

Authors: Jason Brownlee

This book, "Deep Learning With Python," provides a practical guide to developing deep learning models using Theano and TensorFlow with Keras. It's geared towards individuals looking to understand and implement deep learning techniques. The book begins with an introduction to deep learning, outlining the 'wrong way' and setting expectations for the reader. It then delves into the foundational concepts, including the requirements for reading the book and the expected outcomes. A significant portion is dedicated to background information, specifically covering Theano and TensorFlow. The book provides step-by-step instructions on installing both libraries and includes simple examples to illustrate key concepts. It details what Theano is, how to install it, and showcases a basic Theano example. Furthermore, it introduces TensorFlow, explaining its purpose, installation process, and provides initial examples. The book emphasizes hands-on learning and aims to equip readers with the skills to build and deploy deep learning models. The structure is designed for progressive learning, starting with fundamental concepts and gradually introducing more complex techniques. The book's focus on practical implementation using Theano and TensorFlow with Keras makes it a valuable resource for those seeking a hands-on approach to deep learning. The book's v1.7 edition suggests ongoing updates and improvements to the content.
