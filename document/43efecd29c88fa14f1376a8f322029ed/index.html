Big Data Analytics - Overview

Authors: Unknown

This document provides an overview of Big Data Analytics, focusing on the significant increase in data volume coupled with decreasing storage costs. It highlights the challenge of processing the vast amounts of data generated by private companies and research institutions, encompassing user interactions, business data, social media activity, and sensor data from devices like mobile phones and automobiles. Big Data Analytics is presented as the solution to this challenge, involving data collection from diverse sources, transformation of this raw data into consumable formats for analysts, and ultimately, the delivery of valuable data products to organizations. The core of the process lies in converting unstructured data from various sources into usable data products. Furthermore, the document introduces the concept of a data life cycle for Big Data Analytics, drawing parallels with the CRISP-DM methodology. This cycle, though not strictly linear, offers a framework for organizing the work required to derive insights from Big Data. The document explicitly references the Cross Industry Standard Process for Data Mining (CRISP-DM) as a foundational methodology for managing the complexities of Big Data Analytics. The CRISP-DM methodology, a well-established framework, is presented as a useful tool for structuring the entire process, emphasizing the interconnectedness of the various stages involved in extracting meaningful insights from large datasets. The document’s primary goal is to introduce the fundamental concepts and processes associated with Big Data Analytics, particularly the role of data lifecycle management and the application of methodologies like CRISP-DM.
