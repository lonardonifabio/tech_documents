Data Prep with SQL - Quick Reference

This document provides a quick reference guide for data preparation tasks using SQL. It covers several key aspects of data profiling, validation, and cleaning. The document also includes examples of standardization, outlier treatment (quantitatively), and data transformation techniques like casting, replacing, and scaling. The overall goal is to equip users with a concise set of SQL commands and strategies for preparing data for analysis or modeling.
