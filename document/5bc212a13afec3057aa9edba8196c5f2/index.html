Deep Learning with Keras: Cheat Sheet

Authors: RStudio

This document provides a cheat sheet for working with Keras and TensorFlow in R. It focuses on the `keras R` package, which simplifies the use of Keras and TensorFlow within the R environment. The guide outlines key concepts and functionalities, including model creation using the `keras_model_sequential()` function, defining layers, and utilizing the MNIST dataset for demonstration. It details the installation process of the `keras R` package, emphasizing the use of an Anaconda or virtual environment named 'r-tensorflow'. The document covers essential parameters like batch size, epochs, and validation split, alongside concepts like optimizers, loss functions, metrics, and probability prediction. It demonstrates the reshaping and rescaling of data, particularly using the MNIST dataset, and showcases the conversion of labels to categorical format. The guide highlights the importance of setting up the correct environment and installing the necessary libraries to ensure a smooth workflow. The document serves as a quick reference for R users seeking to leverage Keras for deep learning projects.
